# Roadmap

### Hiện trạng

Project đã có:
- trang chủ
- nhiều chuyên mục chính
- template article/category
- build scripts cơ bản

Project còn thiếu:
- tài liệu định hướng chuẩn
- quy ước nội dung và naming rõ ràng
- checklist build/deploy rõ cho người mới
- danh sách việc cần làm theo mức ưu tiên

### Giai đoạn 1: Ổn định nền móng

- Hoàn thiện README và docs nền tảng
- Chuẩn hóa menu, header, footer
- Kiểm tra encoding tiếng Việt toàn bộ project
- Xác định rõ nguồn nào là `source of truth`: Markdown, data hay HTML tay
- Viết checklist build và publish

### Giai đoạn 2: Chuẩn hóa nội dung

- Tạo mẫu bài viết chuẩn
- Tạo mẫu trang danh mục chuẩn
- Chuẩn hóa cách đặt slug, title, description, thumb image
- Bổ sung metadata SEO cơ bản cho các trang

### Giai đoạn 3: Tăng khả năng mở rộng

- Giảm sửa tay lặp lại bằng script/template
- Xây cơ chế sinh page tốt hơn từ Markdown/data
- Tách dữ liệu menu và cấu trúc site ra khỏi HTML cứng nếu cần

### Giai đoạn 4: Hướng sản phẩm

- Tạo landing page cho khóa học đầu tiên
- Tạo page giới thiệu app hoặc công cụ
- Thiết kế luồng chuyển đổi từ bài viết sang sản phẩm

### Việc nên làm tiếp ngay

- Điền đầy đủ phần mô tả trong `docs/PROJECT_BRIEF.md`
- Chọn một quy trình chính: `Markdown first` hoặc `HTML first`
- Chuẩn hóa 1 chuyên mục làm mẫu trước, ví dụ `khoa-hoc`
- Sau khi mẫu ổn, nhân rộng bằng template/script
